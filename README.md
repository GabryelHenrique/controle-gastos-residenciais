# Controle de Gastos Residenciais

Projeto desenvolvido como desafio técnico para uma vaga de estágio.

## Objetivo

Implementar um sistema de controle de gastos residenciais com cadastro de pessoas, cadastro de transações e consulta de totais.

## Tecnologias previstas

- ASP.NET Core Web API
- C#
- React
- TypeScript
- SQLite
- Entity Framework Core

## Status do projeto

Em desenvolvimento.

## Funcionalidades planejadas

- Cadastro de pessoas
- Listagem de pessoas
- Exclusão de pessoas
- Cadastro de transações
- Listagem de transações
- Consulta de totais por pessoa
- Consulta de total geral
